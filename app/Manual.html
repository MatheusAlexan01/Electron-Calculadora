Manual do Usuario

Documentação

Descrição geral (Objetivo): Esse projeto foi desenvolvido no curso técnico de desenvolvimento de sistemas do SENAC, é uma calculadora escolar simples desenvolvida em HTML, CSS e JavaScript. A calculadora permite realizar operações aritméticas básicas, como adição, subtração, multiplicação e divisão. A interface do usuário é intuitiva e fácil de usar, com botões para cada dígito, operadores e funções adicionais como ponto decimal e limpar tudo.

Funcionalidades:

  • Operações Básicas: Adição, subtração, multiplicação e divisão.
  • Números decimal.
  • Limpeza de Dados: Reset da calculadora com o botão "AC".

Como Utilizar:

  1. Inicialização:
    • Ao abrir a página, a calculadora será exibida com uma tela de entrada inicializada com o valor "0".
  2. Entrada de Números:
    • Clique nos botões numéricos (0-9) para inserir números na tela da calculadora.
  3. Operadores:
    • Use os botões de operadores (+, -, *, /) para realizar operações aritméticas. A sequência de uso é: número -> operador -> número -> operador.
  4. Decimal:
    • Utilize o botão de ponto (.) para inserir números decimais.
  5. Resultado:
    • Pressione o botão "=" para calcular o resultado da operação.
  6. Limpar Tudo:
    • Utilize o botão "AC" para resetar a calculadora, retornando a tela de entrada ao valor "0".

Observações:

  • A calculadora suporta operações básicas de adição, subtração, multiplicação e divisão.
  • A calculadora não possui memória para armazenar valores intermediários.
  • O código está bem organizado e segue boas práticas de programação.
